###  Things I've made

| Project | What it is |
|---|---|
| **[termsnap](https://github.com/5tuppl/termsnap)**  | A PTY-based terminal session recorder with a TUI browser. Born from the pain of running `clear` and then wondering what the hell I just did. |
| **[git-brain](https://github.com/5tuppl/git-brain)**  | A pet project that treats your LLM conversation like a git repo — branching, versioning, semantic merge. Lives in my favourite place, the CLI. |
